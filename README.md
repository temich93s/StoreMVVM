# StoreMVVM

Выполнил тестовое задание со **всеми дополнительными пунктами**.

При выполнении задания вел Git по методологии **GitFlow**.

Задание выполнялось с применением следующих технологий:
- **Swift**
- **MVVM + Coordinator + Builder**
- **SwiftUI**
- **Combine**
- **CoreData + URLSession**
